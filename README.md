# TOP1 - Catálogo e Gerenciador de Materiais de Construção

Este é um projeto focado no desenvolvimento de uma aplicação web para a organização, catalogação e consulta de preços de materiais de construção (como itens de drywall e organização de depósitos). 

O repositório também inclui implementações de lógica e algoritmos para fins de estudo de computação.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação das páginas e conteúdos do catálogo.
* **CSS3:** Estilização, design responsivo e organização visual dos elementos.
* **JavaScript:** Criação de funções interativas para a experiência do usuário.

## 🛠️ Funcionalidades do Projeto Web

* Interface limpa e intuitiva para listagem de materiais.
* Organização lógica de insumos e tabelas de preços.
* Scripts focados em facilitar a busca e a usabilidade.

## 📚 Aprendizados

Desenvolver este projeto foi fundamental para consolidar conceitos de:
1. Pensamento computacional e estruturação de algoritmos.
2. Manipulação de elementos na tela com JavaScript puro (Vanilla JS).
3. Versionamento de código e uso profissional do GitHub para portfólio.
